Credit Card Fraud Detection Using Machine Learning

Project type: MSc Machine Learning on Cloud assessment and group project contribution.

This project focused on detecting fraudulent financial transactions from a highly imbalanced credit card dataset. The main challenge was that fraud cases were extremely rare compared with legitimate transactions, so normal accuracy was not enough to evaluate performance.

  • Methods used: Logistic Regression, Random Forest, XGBoost and Neural Network.
  • Imbalance handling: SMOTE, random undersampling and cost-sensitive learning.
  • Evaluation: Recall, Precision, F1-score, ROC-AUC and PR-AUC.
  • Technical contribution: model selection, baseline comparison, cross-validation, hyperparameter tuning and final evaluation.
  • Key learning: PR-AUC and Recall are more meaningful than accuracy when the target class is very rare.

Value for research: This project strengthened my understanding of imbalanced classification, financial risk modelling, reproducible coding and responsible machine learning evaluation.

Big Data Analytics of Drug-Related Deaths

Project type: MSc Big Data Analytics assessment using Python.

This project analysed accidental drug-related death records from 2012 to 2024. The purpose was to identify demographic patterns, substance-related trends and statistically meaningful relationships in public health data.

  • Dataset: 12,964 records and 49 variables related to accidental drug-related deaths.
  • Tools: Python, Pandas, NumPy, Matplotlib, Seaborn and SciPy.
  • Analysis: data cleaning, missing value handling, exploratory data analysis and visualisation.
  • Statistical testing: Chi-square test and Independent Sample t-test.
  • Focus areas: age distribution, sex differences, fentanyl detection and overdose trend patterns.

Value for research: This work shows my ability to connect statistical methods with real public health questions and communicate results through visual evidence.

PythonPublic HealthHypothesis TestingEDA

Business Intelligence Dashboard Using Tableau

Project type: MSc Business Intelligence and visual analytics task.

This project used Tableau to create interactive dashboards from Olympic Games data. The dashboard helped explore medal patterns and relationships across city, season, athlete age and gender.

  • Dashboard focus: medal counts by city, athlete age and medal type, seasonal differences and gender-based medal distribution.
  • Tool used: Tableau for visualisation and dashboard development.
  • Design approach: clear visual layout, interactive filters and connected charts.
  • Outcome: created a user-friendly dashboard that allowed viewers to explore patterns dynamically.

Value for research: This project improved my ability to present analytical findings visually for non-technical audiences and decision-makers.

Machine Learning on Cloud: Technical Evaluation and Reflection

Project type: MSc individual technical evaluation report.

This work critically evaluated the fraud detection machine learning project from a technical and professional perspective. It explained the reasoning behind preprocessing, feature engineering, scaling, imbalance handling, model selection and ethical issues.

  • Technical focus: duplicate removal, log transformation, feature engineering and scaling.
  • Model evaluation: comparison of models and discussion of false positives and missed fraud cases.
  • Professional reflection: teamwork, challenges, lessons learned and ethical considerations.
  • Research awareness: discussed interpretability, fairness, model monitoring and real-world deployment limitations.

Value for research: This experience helped me think beyond model accuracy and consider reproducibility, explainability and responsible AI deployment.

Cloud MLReflectionEthicsModel Evaluation

Meteorological Indicators and COVID-19 Transmission in Bangladesh

Project type: MSc Statistics thesis-based research project.

This project investigated whether meteorological indicators such as temperature and rainfall were associated with COVID-19 transmission in Bangladesh, focusing on Dhaka and Rajshahi regions.

  • Methods used: Spearman correlation, Kendall correlation, OLS regression and Quantile Regression.
  • Variables: confirmed COVID-19 cases, temperature indicators and rainfall.
  • Software/tools: Python, R and SPSS.
  • Research contribution: applied statistical modelling under non-normal distribution conditions.

Value for research: This work strengthened my foundation in applied statistics, environmental statistics and public health data analysis.
